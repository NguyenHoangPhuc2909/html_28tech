                        Bootstrap 4 
1. Giới thiệu và hướng dẫn tích hợp vào web
    1.1. Giới thiệu
    - Bootstrap là một framework của css và javscript (nó là những đoạn code viết sẵn).
    - Định nghĩa sẵn các class trong css và các hàm trong js. Chúng ta chỉ việc sử dụng các class này để tuỳ chỉnh thêm.
    - Giảm thiểu bớt việc viết code, đóng vai trò như bộ khung nền, giúp phát triển web nhanh hơn.
    - Hộ trỡ đa dạng, có sẵn nhiều component (thành phần).
    - Link website: https://getbootstrap.com/
    1.2. Hướng dẫn tích hợp vào web
    - Cách 1: Nhúng link CDN (Link trực tuyến trên server). Xem cách nhúng trên trang getbootstrap.com
    - Cách 2: Tải về và dùng trực tiếp ở local. Tải mã nguồn tại getbootstrap.com
        + Nhúng file css vào trong thẻ head:
        <link rel="stylesheet" href="css/bootstrap.min.css" />
        + Nhúng 2 file js vào cuối thẻ body:
        <script src="js/jquery-3.5.1.slim.min.js"></script> 
        <script src="js/bootstrap.bundle.min.js"></script> 
2. Dựng layout và sử dụng Grid
    2.1. Giới thiệu class .container
    - .container: Giúp căn 2 lề và có độ rộng tối đa là 1140px.
    - Có padding-left và padding-right là 15px.
    - Các kích thước của .container ở các màn hình có kích thước khác nhau:
                    | Extra small |  Small   |  Medium  |  Large   | Extra Large    
                    |  < 567px    | >= 576px | >= 768px | >= 992px | >= 1200px   
    .container      |    100%     |   540px  |  720px   |  960px   |   1140px
    .container-sm   |    100%     |   540px  |  720px   |  960px   |   1140px
    .container-md   |    100%     |   100%   |  720px   |  960px   |   1140px 
    .container-lg   |    100%     |   100%   |  100%    |  960px   |   1140px
    .container-xl   |    100%     |   100%   |  100%    |  100%    |   1140px

    - Ví dụ: Thêm thẻ div sau vào trang html và test thử các màn hình.
    <div class="container" style="background: orange;">Box Container</div>
    2.2. Giới thiệu class .container-fluid
    - .container-fluid: Có độ rộng full màn hình (kéo dài từ trái sang phải)
    - Có padding-left và padding-right là 15px.
    - Các kích thước của .container-fluid ở các màn hình có kích thước khác nhaU:
                        | Extra small |  Small   |  Medium  |  Large   | Extra Large    
                        |  < 567px    | >= 576px | >= 768px | >= 992px | >= 1200px   
    .container-fluid    |     100%    |   100%   |   100%   |   100%   |   100%

    - Ví dụ: Thêm thẻ div sau vào trang html và test thử các màn hình.
    <div class="container-fluid" style="background: orange;">Box Container</div>
    2.3. Giới thiệu media query của Bootstrap cho PC First
        Nhỏ hơn 1200px: @media (max-width: 1199.98px) { /* Code… */ }
        Nhỏ hơn 992px: @media (max-width: 991.98px) { /* Code… */ }
        Nhỏ hơn 768px: @media (max-width: 767.98px) { /* Code… */ }
        Nhỏ hơn 576px: @media (max-width: 575.98px) { /* Code… */ }
    2.4. Grid (Lưới)
    - Grid system (Hệ thống lưới) trong Bootstrap được xây dựng với flexbox.
    - Cho phép tạo tới 12 cột trên một trang.
    - Nếu bạn không muốn sử dụng tất cả 12 cột riêng biệt thì có thể nhóm chúng lại với nhau để tạo ra những cột rộng hơn.
    - Hệ thống lưới này có thể responsive, có nghĩa là các cột có thể tự động sắp xếp lại để phù hợp với kích thước của màn hình.

                            | Extra small |  Small   |  Medium  |  Large   | Extra Large    
                            |  < 567px    | >= 576px | >= 768px | >= 992px | >= 1200px   
    .Max container width    | None (auto) |   540px  |  720px   |  960px   |   1140px
    Class prefix            |    .col-    | .col-sm- | .col-md- | .col-lg- |  .col-xl-

    - Dựng lại một số box trong layout sau sử dụng container, container-fluid, grid và responsive giao diện.
    Link demo: https://bootstrapmade.com/demo/templates/Green/
