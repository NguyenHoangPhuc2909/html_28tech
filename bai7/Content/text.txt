1. Flexbox
1.1. Flexbox là gì?
    - Flexbox layout là một kiểu bố cục trang có khả năng:
        + Tự cân đối kích thước.
        + Thay đổi chiều rộng/chiều cao.
        + Thay đổi thứ tự phần tử.
    - Flexbox có 2 thành phần là:
        + container: là phàn tử cha dùng để bao bọc bên ngoài.
        + item: là phần tử con của container.
1.2. Các thuộc tính của Flexbox
    1.2.1. display flex;
        - Giúp sắp xếp các phần tử một cách dễ dàng, linh hoạt hơn trước.
        - Cú pháp:
            selector {
                display: giá trị;
            }
        - VÍ DỤ:
            .container {
                display: flex; //Giúp các thành phần linh hoạt, CR bằng 100% viewpoint.
            }

            .container {
                display: inline-flex; //Giúp các thành phần linh hoạt, CR bằng với các item bên trong.
            }
    1.2.2. flex-direction
        - Xác định hướng của main-axis để container sắp xếp lại các item.
        - Cú pháp:
            selector {
                flex-direction: giá trị;
            }
        - VÍ DỤ:
            .container {
                display: flex;
                flex-direction: row; //Mặc định, item được sắp xếp theo chiều ngang từ trái sang phải
            }

            .container {
                display: flex;
                flex-direction: row-reverse; //Item được sắp xếp theo chiều ngang, từ phải sang trái (đảo ngược).
            }

            .container {
                display: flex;
                flex-direction: column; //Item được sắp xếp theo chiều dọc, từ trên xuống dưới.
            }

            .container {
                display: flex;
                flex-direction: column-reverse; //Item được sắp xếp theo chiều dọc, từ dưới lên trên.
            }
    1.2.3. Flex-wrap
        - Thuộc tính flex-wrap cho phép item có tự động xuống dòng hay không khi kích thước container thay đổi.
        - Cú pháp:
            selector {
                flex-wrap: giá trị;
            }
            - VÍ DỤ:
            .container {
                display: flex;
                flex-wrap: nowarp; //Mặc định, tất cả các item sẽ nằm trên một dòng.
            }

            .container {
                display: flex;
                flex-wrap: wrap; //Khi tổng chiều rộng các item cộng lại lớn hơn chiều rộng của container thì item sẽ tự động xuống dòng.
            }

            .container {
                display: flex;
                flex-wrap: wrap-reverse; //Tương tư như wrap, nhưng thay vì xuống dòng thì item sẽ tự động nhảy lên trên.
            }
    1.2.4. Justify-content
        - Thuộc tính justify-content để điều chỉnh vị trí bắt đầu và căn chỉnh các item theo trục main-axis.
        - Cú pháp:
            selector {
                justify-content: giá trị;
            }
        - VÍ DỤ:
            .container {
                display: flex;
                justify-content: flex-start; //Mặc định, item sẽ bắt đầu từ lề chính main-start của container.
            }

            .container {
                display: flex;
                justify-content: flex-end; //Item sẽ bắt đầu từ lề chính main-end của container (khác với flex-direction: row-reverse; là đổi hướng hiển thị);
            }

            .container {
                display: flex;
                justify-content: center; //Item sẽ nằm giữa container.
            }

            .container {
                display: flex;
                justify-content: space-between; //Khoảng cách giữa các item bằng nhau. Item đầu tiên và cuối cùng sát với lề.
            }

            .container {
                display: flex;
                justify-content: space-around; //Tương tự space-between, nhưng khác ở chỗ: Khoảng cách giữa item và lề = 1/2 khoảng cách giữa các item với nhau.
            }

            .container {
                display: flex;
                justify-content: space-evenly; //Tương tự space-between, nhưng khác ở chỗ: Khoảng cách giữa item và lề = khoảng cách giữa các item với nhau.
            }
    1.2.5. Align-items
        - Thuộc tính align-items sử dụng để điều chỉnh vị trí bắt đầu và căn chỉnh các item theo trục cross-axis.
        - Dùng để căn chỉnh cho một dòng.
        - Cú pháp:
            selector {
                align-items: giá trị;
            }
        - VÍ DỤ:
            .container {
                display: flex;
                align-items: stretch; // Mặc định, các phần tử sẽ tự động tăng chiều cao để lắp đầy chiều cao của container, nhưung sẽ ưu tiên giá trị height (nếu có).
            }

            .container {
                display: flex;
                align-items: flex-start; //Item sẽ bắt đầu từ lề cross-start của container.
            }

            .container {
                display: flex;
                align-items: flex-end; //Item sẽ bắt đầu từ lề cross-end của container.
            }

            .container {
                display: flex;
                align-items: center; //Item sẽ căn giữa theo chiều của cross-axis.
            }
    1.2.6. Align-content
        - Thuộc tính align-content sử dụng để điều chỉnh vị trí bắt đầu và căn chỉnh các item theo trục cross-axis.
        - Dùng để căn chỉnh cho nhiều dòng.
        - Cú pháp: 
            selector {
                align-content: giá trị;
            }
        - VÍ DỤ:
            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: stretch; //Mặc định, các phần tử tự động tăng chiều cao sao cho lắp đầy container (Vẫn ưu tiên giá trị height nếu có).
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: flex-start; //Item sẽ bắt đầu từ lề chứa cross-start của container.
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: flex-end; //Item sẽ bắt đầu từ lề chứa cross-end của container.
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: center; //Item sẽ nằm giữa container căn theo cross-axis.
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: space-around; //Tương tự space-between, nhưng khác ở: Khoảng cách giữa item và lề = 1/2 khoảng cách giữa các item với nhau.
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: space-between; //Khoảng cách các item bằng nhau. Item đầu tiên và cuối cùng sát với lề.
            }

            .container {
                display: flex;
                flex-wrap: wrap;
                align-content: space-evenly; //Tương tự space-between, nhưng khác ở: Khoảng cách giữa item và lề = khoảng cách giữa các item với nhau.
            }
    1.2.7. Align-self
        - Tương tự như align-items nhưng là áp dụng cho một item nào đó.
        - Cú pháp:
            selector {
                align-self: giá trị;
            }
        - VÍ DỤ:
            .item:nth-child(2){
                align-self: auto; //Mặc định
            }

            .item:nth-child(2){
                align-self: stretch; //Các phần tử sẽ tự động tăng chiều cao để lắp đầy chiều cao của container, nhưng sẽ ưu tiên giá trị height (nếu có). 
            }

            .item:nth-child(2){
                align-self: flex-start; //Item sẽ bắt đầu từ lề cross-start của container.
            }

            .item:nth-child(2){
                align-self: flex-end; //Item sẽ bắt đầu từ lề cross-end của container.
            }

            .item:nth-child(2){
                align-self: center; //Item sẽ căn giữa theo chiều của cross-axis
            }
    1.2.8. Order
        - Thuộc tính Order dùng để sắp xếp lại vị trí của các item.
        - Cú pháp:
            selector {
                order: giá trị; //Mặc định là 0
            }
        - Trong đó: giá trị >= 0
